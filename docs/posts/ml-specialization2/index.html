- Activation function
    
    **Alternative to sigmoid function**
    
    ![Linear - Sigmoid - ReLU](Advanced%20learning%20algorithm%20@November%2024,%202023%20f830f807954140d0898eb9ae02adfab7/Untitled%2029.png)
    
    Linear - Sigmoid - ReLU
    
    **How to choose activation function for neural network?**
    
    ***For output layer:***
    
    ![Untitled](Advanced%20learning%20algorithm%20@November%2024,%202023%20f830f807954140d0898eb9ae02adfab7/Untitled%2030.png)
    
    ***For hidden layer:***
    
    ![Untitled](Advanced%20learning%20algorithm%20@November%2024,%202023%20f830f807954140d0898eb9ae02adfab7/Untitled%2031.png)
    
    *here in sigmoid function there are two flat zone. using sigmoid function as activation function can result in more flat regions in cost function which results in slower gradient descent.*
    
    ***Summary:***
    
    For binary classification → Sigmoid function
    
    For regression (+/-) → Linear function
    
    For regression (0 or +) → ReLU
    
    ***Code implementation:***
    
    ```python
    from tf.keras.layers import Dense
    model = Sequential([
            Dense(units = 25, activation = 'relu'), #layer1
            Dense(units = 15, activation = 'relu'), #layer2
            Dense(units = 1, activation = 'sigmoid' or 'linear') #layer3
    ])
    ```
    
    **Why do we need activation functions?**
    
    if we only use linear function for each layer of neural network then the neural network is equivalent to a linear regression 
    
    ![neural network → linear regression](Advanced%20learning%20algorithm%20@November%2024,%202023%20f830f807954140d0898eb9ae02adfab7/Untitled%2032.png)
    
    neural network → linear regression
    
    ***Don’t use linear activation in hidden layers:***
    
    ![neural network → logistic regression](Advanced%20learning%20algorithm%20@November%2024,%202023%20f830f807954140d0898eb9ae02adfab7/Untitled%2033.png)
