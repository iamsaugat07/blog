Linear regression model
  • What is linear regression model?
    • fit a line or model to the training data such that it gives minimum error (cost function). model parameters are adjusted to get the desired model.
  • Example: House sizes and prices
  • What is cost function?
    • it helps us visualize how well our model is predicting by comparing the value predicted by the model to the actual target data in the training set.
    • cost function may vary according to the model
    • for example: Generally for linear regression model, mean squared error is used as cost function
